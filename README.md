# Altare Pacis — Static Site (Red/Black/Yellow)

В этом архиве готовый статический сайт для загрузки на GitHub Pages или Netlify.

Файлы:
- `index.html` — главная страница (красно-чёрно-жёлтая тема)
- `store.html` — страница магазина STL с примерами товаров
- `styles.css` — стили
- `logo.svg` — логотип (вставьте ваш, если нужно)
- `files/` — (папка для STL — добавьте свои файлы в папку `files/`)

Инструкция для GitHub (на телефоне):
1. Скачайте ZIP и распакуйте его в файловом менеджере телефона.
2. В репозитории GitHub нажмите **Add file → Upload files**, выберите ВСЕ файлы из распакованной папки (index.html, store.html, styles.css, logo.svg и папку files).
3. Закоммитьте изменения.
4. В Settings → Pages включите GitHub Pages (branch: main, folder: /root).
5. Через минуту сайт будет доступен по ссылке типа `https://<username>.github.io/<repo>`.

Инструкция для Netlify:
1. На Netlify используйте **Add new site → Deploy manually** и загрузите этот ZIP (Netlify распакует содержимое).
2. Сайт будет опубликован автоматически.

Оплата:
- В кнопках товара стоит mailto: ссылка (чтобы получать запросы).
- Для реальной оплаты: получите ссылку PayPal/Stripe и замените href у кнопки «Купить» на неё.

Если хотите, я могу:
- Вставить ваши STL-файлы в папку `files/`.
- Подключить PayPal/Stripe к кнопкам.
- Изменить логотип на ваш вариант.
